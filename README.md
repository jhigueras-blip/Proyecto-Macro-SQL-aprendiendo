# Proyecto-Macro-SQL-aprendiendo
Proyecto de análisis Macroeconómico de 10 economías para aprender a usar SQL. 

## Descripción
Este proyecto analiza el comportamiento macroeconómico de 10 países durante la última década (2012-2023), con foco especial en la comparación entre Chile y 9 economías que comparten ciertas similitudes.

## Herramientas Utilizadas
* SQL (SQLite)
* Funciones de agregación, Window Functions (LAG, RANK) y lógica condicional (CASE).

## Hallazgos Principales
1.  **Resiliencia de Chile:** A pesar del peak inflacionario de 2022, Chile mantiene un puntaje de estabilidad de 9/10.
2.  **Comparativa:** Malasia se perfila como la economía más estable del grupo de estudio.
